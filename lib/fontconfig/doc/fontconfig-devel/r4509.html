FcStrFree

Name

FcStrFree -- free a string

Synopsis

#include <fontconfig.h>

void FcStrFree(FcChar8 *s);

Description

This is just a wrapper around free(3) which helps track memory usage of strings within the fontconfig library.

Version

Fontconfig version 2.6.0
