Name

FcStrSetDel -- delete from a string set

Synopsis

#include <fontconfig.h>

FcBool FcStrSetDel(FcStrSet *set, const FcChar8 *s);

Description

Removes s from set, returning FcTrue if s was a member else FcFalse.

Version

Fontconfig version 2.6.0
